O que o bot faz

Fluxo completo de atendimento automatizado para farmácia.

✓ Recebe mensagens via webhook Z-API
✓ Gerencia sessões por telefone (FSM)
✓ Catálogo com 6 categorias e 14 produtos
✓ Carrinho, endereço e pagamento
✓ Respostas dinâmicas com Groq
✓ Transferência para atendente humano
✓ Métricas enviadas ao dashboard
